# Docker golang image

Build to Docker Image
OS selected [CentOS](https://registry.hub.docker.com/_/centos/)
With [golang](http://golang.org/)

## Usage

```sh
$ docker pull nishigori/golang
```

---

## Development

### Requirements

* docker ~> 1.0.0
* boot2docker ~> 1.0.0 (only Mac OS)

```
$ docker build -t {APP_NAME} .
```
